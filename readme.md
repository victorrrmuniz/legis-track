# Legistrack

Project for the Quorum Coding Challenge

## Technologies

- **Frontend**: React + Vite
- **Backend**: C#

## Requirements

- **Node.js**
- **.NET SDK**


## How to run the project

### Frontend
- npm install

- npm run dev

- the project will run in http://localhost:5173/

### Backend
- dotnet build

- dotnet run
